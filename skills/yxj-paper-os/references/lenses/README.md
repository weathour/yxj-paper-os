# Canonical Lens Registry

Lenses are conditional reasoning modules, not paper types or workflow stages. Activate
only the lenses whose triggers affect the requested scope. Every lens uses the same
four owner gates: `scientific_commitment`, `argument_spine`,
`material_local_tradeoff`, and `deliberate_divergence`. Routine grounded reversible
design remains agent-decided.

## Lens Registry

| Lens ID | Module path | Primary focus | Typical activation |
|---|---|---|---|
| contribution-evidence-forms | contribution-evidence-forms.md | contribution realization and evidence ceiling | contribution, comparator, or claim-role ambiguity |
| research-design-validity | research-design-validity.md | validity threats and causal/non-causal boundaries | study-design or inference-risk surface |
| evidence-results-statistics | evidence-results-statistics.md | result functions, uncertainty, displays, and adverse findings | result, metric, figure, or table surface |
| literature-differentiation | literature-differentiation.md | source roles, citation functions, contrast, and dual-role firewall | related-work or differentiation surface |
| reproducibility-governance | reproducibility-governance.md | method/reporting/artifact/availability handoff | reproducibility or required-statement surface |
| argument-language-visual | argument-language-visual.md | paragraph functions, controlled frames, language, visuals, closure | argument, wording, caption, or accessibility risk |
| venue-template | venue-template.md | semantic-reading-first template design with optional analyzer | eligible templates can materially change a requested surface |

A generic fallback pointer is valid only as
`lens:<Lens ID>#<real-h2-or-h3-slug>` through this unique mapping. H1-only or
invented slugs are invalid. The `venue-template` lens uses grounded model-semantic
deep reading as the primary path. The deterministic analyzer is optional, answers
only a declared measurable question, and is never a readiness prerequisite.
