# yxj-paper-plan

Create paper plans only after source-map and evidence boundaries are known. Use `$oh-my-codex:ralplan` when architecture, tradeoffs, or test-shape consensus is needed.

## Required planning artifacts

- confirmed motivation or explicit blocker;
- contribution map;
- section blueprints;
- writing rationale matrix with evidence/citation anchors;
- branch owner map;
- validator list and expected failure modes;
- state transition plan.

Do not route to writing if motivation is unconfirmed or evidence bank is missing for claim-bearing sections.
