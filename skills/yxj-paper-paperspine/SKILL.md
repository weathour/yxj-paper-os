# yxj-paper-paperspine

Translate research and evidence into a manuscript spine.

## Required sequence

1. Confirm motivation.
2. Build contribution map.
3. Build section blueprints.
4. Build writing rationale matrix.
5. Assign branch owners and validators.
6. Only then route bounded drafting/revision tasks.

Do not write claim-bearing prose from a shallow rationale matrix.
