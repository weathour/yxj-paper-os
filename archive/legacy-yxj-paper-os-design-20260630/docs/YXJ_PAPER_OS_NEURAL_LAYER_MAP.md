# yxj-paper-os Neural-Layer Material Map

This map reorganizes every template/material from `$yxj-paper-os` into a neural-network-like paper-production hierarchy.

Core reinterpretation:

```text
initial materials
  -> research/template/evidence understanding
  -> claim and reader-control layers
  -> writing and figure production layers
  -> review/loss layers
  -> local backflow / repair
  -> export / handoff
```

Departments are not runtime loops here. They are removed from the main cognitive flow and retained only as sidecar governance metadata.

## Coverage

- yxj-paper-os template/material files covered: **70 / 70**.
- Main forward-paper-production layers use the semantic, evidence, narrative, writing, figure, review, backflow, and export materials.
- Sidecar regions contain governance, PMO, department, plugin-absorption, and post-paper derivative materials that do not belong to the manuscript cognition forward pass.

## Important interpretation

The old `$yxj-paper-os` analysis objects are mostly valuable. The new PPG model changes their organization:

```text
old: department -> internal loop -> report
new: material node -> transform -> candidate -> validator -> graph commit/backflow
```

Thus every material in this inventory can later become either a graph node, a transform output, a validator report, or a sidecar governance node.
